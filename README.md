# agua-kmzero
Repository to AguaKMZERO.

**Installing Vue**
npm install -g @vue/cli
vue create agua-kmzero-interface
cd agua-kmzero-interface

**Firebase Config**
npm install firebase
// Initialize Firebase
const app = initializeApp(firebaseConfig);
const analytics = getAnalytics(app);
